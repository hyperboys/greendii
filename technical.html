🔧 For Developers & Engineers

System Overview

ระบบ Sales Workflow Management ถูกออกแบบเป็น Web-based Application แบบ Full-stack ที่ใช้ Modern Technology Stack เพื่อรองรับการทำงานของบริษัท กรีนส์ดี จำกัด

🎯 Core Features

  • User Authentication & Authorization
  • Document Management (Quotation, WO, HO, PR)
  • Auto Number Generation
  • Multi-step Approval Workflow
  • PDF Generation & Email Integration
  • File Upload & Storage
  • Role-based Access Control (RBAC)
  • Audit Trail & Logging
  • Dashboard & Reporting
  • Notification System

👥 User Roles

  • Sales (เซลล์) — สร้างและแก้ไขเอกสาร
  • Sales Manager — อนุมัติเอกสารระดับ Manager
  • Admin Manager — อนุมัติด้านการบริหาร
  • Project Manager — อนุมัติด้านโครงการ
  • Managing Director — อนุมัติสูงสุด
  • Procurement (จัดซื้อ) — จัดการเอกสารหลังอนุมัติ
  • Factory (โรงงาน) — รับงานและปิด Work Order
  • Admin — จัดการระบบและผู้ใช้

Approval Flow Implementation

🔄 ลำดับการอนุมัติเอกสาร

เอกสารจะถูกส่งต่ออัตโนมัติตามลำดับ ผู้อนุมัติแต่ละคนสามารถ Approve หรือ Reject ได้

⚙️ Implementation Logic

  • ใช้ State Machine Pattern
  • บันทึก Log ทุก Step ใน approval_logs
  • ตรวจสอบ Role ก่อนอนุมัติ
  • ส่ง Notification อัตโนมัติ (อีเมล + LINE)
  • เมื่อ Approve แต่ละ Step → Push LINE ถึงผู้อนุมัติคนถัดไป
  • รองรับ Skip Step (ถ้าผู้อนุมัติไม่ว่าง)
  • Reject → ส่งกลับเซลล์คนที่ 1
  • Timeout Alert (เกิน 3 วัน ไม่อนุมัติ)

💡 Business Rules

  • เซลล์คนที่ 1 สร้างเอกสาร → ส่งเซลล์คนที่ 2 เช็ค
  • เซลล์คนที่ 2 อนุมัติ → ส่ง Sales Manager
  • Sales Manager อนุมัติ → ส่งไปตามสายจนถึง Managing Director
  • Managing Director อนุมัติ → ส่งจัดซื้อ
  • จัดซื้อตัดราคาขาย → ส่งโรงงาน
  • โรงงานปิดงาน → สิ้นสุด Workflow
